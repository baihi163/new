## 2. 实验思考与总结

### Q1: 学习率 (Learning Rate) 对训练的影响
我在代码中修改 `LEARNING_RATE` 进行了实验，观察到以下现象：
*   **当学习率过大时（如 LR=1.0）**：
    Loss 曲线无法下降，甚至出现 NaN (数值溢出) 或剧烈震荡。这是因为步长太大，导致优化器在最小值附近“反复横跳”，无法收敛。
*   **当学习率过小时（如 LR=0.00001）**：
    Loss 下降极其缓慢，训练了 1000 轮后，红线（预测线）可能还是一条直线，几乎没有弯曲。这是因为步长太小，还没走到山谷底部训练就结束了。

### Q2: 关于过拟合 (Overfitting)
*   **如何判定？**
    如果红线不再是平滑的波浪，而是为了强行经过每一个噪点（灰点），变得歪歪扭扭、甚至出现尖锐的折线，这就是过拟合。
*   **如何解决？**
    可以通过 **减少隐藏层神经元数量** (比如把 `HIDDEN_SIZE` 从 20 改为 5)，或者减少网络层数，或者增加正则化项 (Weight Decay) 来解决。
